# Freelancing Platform

## Project Overview
This project is a web-based freelancing platform where companies can post jobs and employees can apply for jobs online.

## Features

### Admin
- Manage users
- Monitor transactions
- View reports

### Company
- Post jobs
- Hire employees
- Manage company profile

### Employee
- Apply for jobs
- View jobs
- Get ratings

## Technologies Used
- C#
- ASP.NET
- SSMS

## Database Tables
- Users
- Roles
- Companies
- Jobs
- Applications
- Transactions
- Ratings
- Reports

## How to Run
1. Open project in Visual Studio
2. Connect database
3. Run the application
